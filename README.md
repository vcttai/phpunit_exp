# PHPUnit experimental project

## How to run test

### Run all test cases
- Clone this project
- Install dependencies
```
composer install
```
- Run test
```
composer test
```

### Run a specific test case
- Run as following command:
```
./vendor/bin/phpunit <filename> 

e.g 
./vendor/bin/phpunit test/00_hello_phpunit/HelloPHPUnitTest.php 
```
